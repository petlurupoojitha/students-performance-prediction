

# Student Performance Prediction using Machine Learning

## Overview

This project focuses on predicting student exam scores using Machine Learning techniques. The model analyzes important factors such as **study hours, attendance, and previous marks** to estimate the expected performance of students. The goal is to demonstrate how data preprocessing and regression models can help understand patterns in academic performance.

## Problem Statement

Predicting student performance manually can be difficult and may not always be accurate. Educational institutions need a data-driven approach to understand how different factors influence student exam results. This project aims to build a **machine learning model that predicts student exam scores** using study hours, attendance percentage, and previous marks.

## Solution

To solve this problem, a **Linear Regression model** is implemented. The dataset is first cleaned and preprocessed to remove inconsistencies. Important features such as study hours, attendance, and previous marks are used to train the model. The trained model then predicts student exam scores based on these input factors.

## Tools & Technologies

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Linear Regression Algorithm

## Workflow

1. Data Collection
2. Data Preprocessing using Pandas
3. Exploratory Data Analysis and Visualization
4. Feature Selection (study hours, attendance, previous marks)
5. Model Training using Linear Regression
6. Model Evaluation and Prediction
7. Visualization of Results using Matplotlib

## Result

The model successfully predicts student exam scores based on the given input features. The analysis shows that **study hours and attendance have a strong influence on exam performance**. This project demonstrates the practical use of machine learning for educational data analysis.
