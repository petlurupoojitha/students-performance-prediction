
# Student Performance Prediction using Machine Learning

This project predicts student exam scores using Machine Learning based on factors such as study hours, attendance, and previous marks. The model uses **Linear Regression** to analyze patterns in student data and estimate expected performance. Data preprocessing and visualization are done using **Pandas and Matplotlib**. The project helps understand basic **regression techniques and data analysis** in Python. Technologies used include **Python, Pandas, Scikit-learn, and Matplotlib**.
